## 可用脚本

### `yarn complie`
yarn run tsc 的 alias, 编译ts

### `yarn complie --watch`
yarn run tsc --watch 的 alias, 以监视方式编译ts


### `yarn lint`
yarn run eslint

### `yarn start`
node ./dist/index.js 运行脚本

#### -c --config <path> 指定设置文件，默认设置在./src/config.ts中或./dist/config.js中, 不建议再自定设置脚本.

#### -s --silent 静默运行
